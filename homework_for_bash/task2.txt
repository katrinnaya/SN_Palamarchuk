Задание 2. Пункт 2
Объясните, почему изменения переменной PATH, сделанные через терминал, временные,
и предложите способ сделать их постоянными. 
Добавьте команду в файл .bashrc и продемонстрируйте, как перезапустить терминал для применения изменений.

Объяснение:
Изменения переменных окружения, напр. переменной PATH, сделанные непосредственно в терминале, являются временными.
Они действуют только до тех пор, пока открыт текущий сеанс терминала.
Когда мы закрываем терминал, изменения теряются.
Соответственно, при следующем открытии терминала переменные возвращаются к своим исходным значениям.

Это связано с тем, что изменения переменных окружения применяются только к текущему процессу и его подпроцессам.
При закрытии терминала процесс завершается, и изменения исчезают.
Чтобы сделать изменения постоянными, нужно внести соответствующие строки в файл конфигурации, например, .bashrc или .profile.

Постоянное изменение переменной PATH:
1. Открываем файл .bashrc в домашнем каталоге
2. Добавляем строку для обновления переменной PATH в конец файла
3. Сохраняем изменения в редакторе, закрываем редактор
4. Перезагружаем файл .bashrc или перезапускаем терминал
После выполнения шагов изменения переменной PATH станут постоянными и будут применяться при каждом новом запуске терминала.

В качестве примера прикрепляю код:
export PATH="$PATH:/usr/local/bin" #Напр., добавилю директорию /usr/local/bin в мой PATH. В файле .bashrc добавляю эту строку
source ~/.bashrc #Ввожу код в терминале для перезагрузки файла
